# File manifest

## Notebooks

- `notebooks/01_build_public_data_package_2023.ipynb`
- `notebooks/02_reproduce_screening_outputs_2023.ipynb`

## Scripts

- `scripts/01_build_public_data_package_2023.py`
- `scripts/02_reproduce_screening_outputs_2023.py`

The script files are plain-Python counterparts of the notebooks.

## Data package

- `data/packages/CBEC_EU27_revised_data_package_2023.zip`
- `data/processed/` contains the extracted CSV/XLSX files from that package.

## Output package

- `outputs/packages/CBEC_EU27_revised_publication_outputs_2023.zip`
- `outputs/tables/` contains generated tables and workbook outputs.
- `outputs/figures/` contains generated figures.
- `outputs/source_data/` contains figure source data.
- `outputs/auxiliary/` contains additional generated support files.
