# Reproducibility notes

Recommended route:

1. Run `notebooks/01_build_public_data_package_2023.ipynb` in Google Colab.
2. Download `CBEC_EU27_revised_data_package_2023.zip`.
3. Run `notebooks/02_reproduce_screening_outputs_2023.ipynb` in Google Colab.
4. Upload the data-package ZIP when prompted.
5. Download `CBEC_EU27_revised_publication_outputs_2023.zip`.

The repository already includes generated packages and extracted outputs for inspection. Exact values can depend on the availability and current state of external public-data services if notebook 01 is rerun later.
